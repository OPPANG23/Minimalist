### PARABOLA – Desktop Website Replica

This is a static single-page site replicating the provided desktop layout.

#### Getting started
- Open `index.html` in any modern browser.

#### Structure
- `index.html`: Markup for sections: New Collection, About, Lookbook, Footer.
- `styles.css`: Dark theme, condensed headings, product grid, and large brand wordmark.
- `assets/`: Lightweight SVG placeholders for imagery.

#### Notes
- Fonts are loaded from Google Fonts: Bebas Neue for headlines, Inter for body.
- Grid will adapt on narrow viewports, but design targets desktop as requested.
