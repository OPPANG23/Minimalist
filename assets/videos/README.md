# Videos

Place your MP4 files here and keep the names used in `index.html`:

- `forest.mp4`
- `waterfall.mp4`
- `meadow.mp4`
- `shore.mp4`

Recommendations:

- Encode H.264 in MP4 (Main profile), 1080p or 1440p, 24–30 fps
- Keep bitrate 6–10 Mbps for quality, lower for faster loads
- Start videos with a still frame for cleaner lazy-load transitions
- Keep short loops (10–30s) for calm, ambient playback
